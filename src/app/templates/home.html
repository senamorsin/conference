What it does well

  • Live fingerspelling at ~8 FPS using MediaPipe hand landmarks.
  • Single-sign classification with calibrated confidence and explicit rejection.
  • Multi-word transcripts from a single recorded clip with per-segment time ranges.

What it can't do (yet)

  • Translate arbitrary open-vocabulary ASL — the word model knows 50 signs.
  • Handle continuous signing without pauses — sequence mode assumes brief gaps between signs.
  • Replace a human interpreter. This is an MVP for demonstrating the pipeline.
